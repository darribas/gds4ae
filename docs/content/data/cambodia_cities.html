Thin#

We are interested in Cambodian cities only:

khm = db.query("CTR_MN_NM == 'Cambodia'")

And only on a subset of the attributes available:

vars_to_keep = [
    "ID_HDC_G0", # Unique ID
    "UC_NM_MN",  # City name
    "NTL_AV",    # Mean night time light
    "E_GR_AV14", # Mean greenness 2014
    "geometry"
]
